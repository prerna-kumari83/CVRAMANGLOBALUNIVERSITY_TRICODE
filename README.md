# 👗 Fashion Tips Popup (Myntra-Inspired UI)

An animated, modern fashion tips popup built using **HTML, CSS, JavaScript, and GSAP**.
This project displays random styling tips in a visually appealing **glassmorphism UI**, inspired by modern e-commerce platforms.

---

## ✨ Features

* 🎬 Smooth animations using GSAP
* 🎨 Glassmorphism (blur + gradient UI)
* 🖼️ Dynamic image + text updates
* 🔁 Random fashion tips on each popup
* ⏱️ Auto ظهور (after 3 seconds) & auto close
* ❌ Manual close button for better UX
* 💎 Clean, modern, responsive design

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Flexbox + Glassmorphism)
* JavaScript (DOM Manipulation)
* GSAP (Animation Library)

## 🎯 How It Works

* A popup appears after **3 seconds**
* A **random fashion tip** and image are displayed
* GSAP handles smooth animations
* Popup automatically disappears after **10 seconds**
* Users can also close it manually
